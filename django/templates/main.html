Confusion Matrix(혼동행렬)

Confusion Matrix 설명

혼동 행렬은 모델의 예측 성능을 평가하는 데 사용됩니다. 행렬의 각 요소는 다음과 같은 의미를 가집니다:

  • True Positive (TP): 실제로 이탈한 고객을 정확히 이탈한 것으로 예측한 경우 (925)
  • False Positive (FP): 실제로 이탈하지 않은 고객을 이탈한 것으로 잘못 예측한 경우 (111)
  • True Negative (TN): 실제로 이탈하지 않은 고객을 정확히 이탈하지 않은 것으로 예측한 경우 (183)
  • False Negative (FN): 실제로 이탈한 고객을 이탈하지 않은 것으로 잘못 예측한 경우 (190)

혼동 행렬을 통해 모델의 정확도, 정밀도, 재현율 등의 다양한 성능 지표를 계산할 수 있습니다.

Normalized Confusion Matrix(정규화된 혼동 행렬)

Normalized Confusion Matrix 설명

정규화된 혼동 행렬은 각 클래스의 예측 비율을 시각화한 것입니다. 각 셀의 값은 해당 클래스의 예측 비율을 나타냅니다.

  • True Positive Rate (TPR): 실제 양성 샘플 중 양성으로 예측한 비율 (재현율)
  • False Positive Rate (FPR): 실제 음성 샘플 중 양성으로 잘못 예측한 비율

정규화된 혼동 행렬을 통해 모델의 예측 성능을 더 명확하게 이해할 수 있습니다.

ROC Curve

ROC Curve 설명

ROC 곡선은 모델의 분류 성능을 평가하는 데 사용됩니다. 곡선의 각 요소는 다음과 같은 의미를 가집니다:

  • True Positive Rate (TPR): 실제 양성 샘플 중 양성으로 예측한 비율 (재현율)
  • False Positive Rate (FPR): 실제 음성 샘플 중 양성으로 잘못 예측한 비율

ROC 곡선 아래의 면적 (AUC)은 모델의 분류 성능을 나타내며, 1에 가까울수록 성능이 좋음을 의미합니다.

Precision-Recall Curve

Precision-Recall Curve 설명

Precision-Recall 곡선은 모델의 분류 성능을 평가하는 데 사용됩니다. 곡선의 각 요소는 다음과 같은 의미를 가집니다:

  • Precision: 양성으로 예측한 샘플 중 실제 양성인 샘플의 비율
  • Recall: 실제 양성 샘플 중 양성으로 예측한 샘플의 비율 (재현율)

Precision-Recall 곡선 아래의 면적 (AUC)은 모델의 분류 성능을 나타내며, 1에 가까울수록 성능이 좋음을 의미합니다.

Softmax Probability Distribution

Softmax Probability Distribution 설명

소프트맥스 확률 분포는 모델의 예측 확률을 시각화한 것입니다. 각 확률 값은 모델이 해당 클래스에 속할 확률을 나타냅니다.

  • Softmax: 각 클래스에 대한 확률을 계산하는 함수
  • Probability Distribution: 각 클래스에 대한 예측 확률의 분포

소프트맥스 확률 분포는 모델의 예측이 얼마나 확신에 차 있는지를 나타내며, 확률 값이 높을수록 모델이 해당 클래스에 속할 가능성이 높다고 판단한 것입니다.
